# FAQ

## Is this an official project of upstream services?

No. This launcher is an independent integration layer.

## Which Python version should I use?

Python 3.13.x is recommended for the AniWorld part of this setup on Windows.

## Which Node.js version should I use?

Node.js LTS is recommended. This project was tested with Node.js 24.x.

## Where should I report problems?

Use GitHub Issues in this repository.

## Is this project legal everywhere?

Laws vary by country. You are responsible for legal compliance in your jurisdiction.

## Was AI used to build this?

Yes. Parts of the implementation and documentation were created with AI assistance.

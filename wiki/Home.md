# AniWorld / AniWatch Launcher Wiki

Welcome to the full documentation for the launcher.

## What This Project Does

This project provides a unified Windows launcher for:

- AniWorld Downloader (Web UI)
- AniWatch API
- AniWatch UI

The launcher can install/repair dependencies and start services with one menu.
